# COMP2090SEF_GRP_74
COMP2090SEF Course Project\
 \
User Guide:\
 \
A Movie CD rental management system built with Python OOP.\
 \
Add, View, Borrow and Return Movie CDs\
Customer management\
View most popular CDs (using MaxHeap)\
Sort CDs by popularity (using Heap Sort)\
\
cd.py is the Movie CD class and abstract base class\
customer.py is the Customer class\
rental_system.py is the Main system logic + MaxHeap + Heap Sort\
main.py is the Program entry point\
